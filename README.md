# Symfony CRUD Application

A simple **CRUD (Create, Read, Update, Delete)** web application built with the **Symfony Framework**.  
This project demonstrates the use of Symfonyâ€™s Form, Validation, Doctrine ORM, and Twig templating system.

---

##  Features

- Create new products  
- Display a list of products  
- View detailed information about a product  
- Edit existing products  
- Delete products with confirmation  
- Flash messages for user feedback  
- Validation on both client and server sides

---

##  Technologies Used

- **Symfony**
- **Doctrine ORM**
- **Twig** (templating engine)
- **MySQL** (database)
